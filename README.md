### Setup
You will need to get ESPN_S2 and SWID cookies thru dev tools in your browser. You will also need your League ID found in ESPN

### Weekly Scores
Pulls from ESPN API and creates several charts using input year data

Ensure `year`, `week_max`, and `current_week` are set appropriately

### Historical Draft Data
Pulls draft data between `year_end` and `year_start` and provides charts

### Actual vs Optimal Scores
Shows the actual scores by team vs the possible optimal scores for given `season` (year) and `week`
